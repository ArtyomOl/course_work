# Система семантического поиска

Программная система для семантического поиска в текстовых документах с использованием алгоритма TF-IDF.

## Требования

- Python 3.11 или выше
- PyQt5

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd course_work
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Инициализируйте систему (только при первом запуске):
```bash
python init_index.py
```

## Запуск

Запустите приложение командой:
```bash
python main_window.py
```

Или на macOS/Linux:
```bash
python3 main_window.py
```

## Структура проекта

```
course_work/
├── main_window.py              # Главное окно приложения
├── text_reader_form.py         # Форма для чтения и редактирования документов
├── backend/
│   └── core/
│       ├── document_manager.py # Управление документами
│       ├── indexer.py          # Индексация документов
│       ├── search.py           # Поисковый движок
│       ├── recommender.py      # Рекомендательная система
│       ├── text_preprocess.py  # Предобработка текста
│       ├── query.py            # Обработка запросов
│       ├── index.py            # Работа с индексом
│       ├── search_history.py   # История поиска
│       ├── search_result.py    # Результаты поиска
│       └── database.py         # Работа с базой данных
└── data/
    └── documents/              # Текстовые документы
```

## Функциональность

- Семантический поиск по документам
- Фильтрация результатов по ключевым словам
- Просмотр и редактирование документов
- Просмотр похожих документов
- История поисковых запросов
- Рекомендации на основе истории поиска
- Добавление новых документов
- Удаление документов

## Технологии

- Python 3.11
- PyQt5 (графический интерфейс)
- SQLite (база данных)
- TF-IDF (векторизация текста)
